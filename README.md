# LA Airbnb Analysis

This repo contains a data science project analyzing Airbnb listings in Los Angeles for ST 625.

## Contents
- `LA_Airbnb.ipynb`: main analysis notebook.
- `index.qmd`, `viz.qmd`, `interactive.qmd`: Quarto website pages.
- `_quarto.yml`: Quarto website configuration.
- `requirements.txt`: Python packages needed to run the analysis.
